# Installation and usage

In this folder you will find two python scripts and one bash shell script.
The bash script will only run on Linux/MacOS (or Windows with WSL).

What are they:
  - max_speed_db_cli.py:  command line python utility to create a local sqlite database for the country you specify.
  - osm_to_speed_db_ui.py:  Graphical UI python utility to create a local sqlite database for the country you specify.
  - create_sqlite_countries.sh:  A bash shell script to create all country databases for a specific continent using the "max_speed_db_cli.py" command line script.
